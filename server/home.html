Welcome to the most basic pokedex API in existence

Made with express - Part of a Codepath course

API Endpoints

GET All Pokemons

GET /pokemons

Retrieve a list of all pokemons in the system.

Optional Parameters

page Page list of pokemons relative to the number of pokemons retrieved

limit Size of the list of pokemons to be retrieved. Default is 10

GET Pokemon By ID

GET /pokemons/:id

Retrieve all information of a pokemon by its id
